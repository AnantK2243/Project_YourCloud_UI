Node Setup Tutorial

Step 1: Install the Storage Client (Requires Rust Compiler)

Linux Installation

# Clone the repository
git clone https://github.com/AnantK2243/Project_YourCloud_Client.git --depth 1

# Navigate to the extracted directory
cd Project_YourCloud_Client

# Ensure you have Rust installed (https://www.rust-lang.org/tools/install)
# If not, install Rust using the command below:
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Follow the on-screen instructions to complete the installation

# Build the project for release
cargo build --release

# Run the installation script
sudo ./scripts/install.sh

# The client will be installed to /usr/local/bin/project_yourcloud

Step 2: Configure Your Node

After registering your node in the dashboard, configure it using the config file:

Configuration File Location

  • Linux/macOS: ~/.config/Project_YourCloud/config.toml

Step 3: Start the Storage Node

Once configured, start your storage node:

Manual Start

# Start the storage client
project_yourcloud

Run as System Service (Linux)

# Enable and start the systemd service
sudo systemctl enable project_yourcloud_daemon
sudo systemctl start project_yourcloud_daemon

# Check service status
sudo systemctl status project_yourcloud_daemon

Step 4: Verify Connection

Go back to your dashboard to verify that your node appears as "online" and is ready to store files.

Troubleshooting

  • Ensure your firewall allows connections on the configured port
  • Check that the storage path exists and has proper permissions
  • Verify your Node ID and Auth Token are correctly entered
